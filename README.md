# ENG // Utility Console

A minimalist engineering utility tool for image processing and data export.

## Features

- **Image Workflow**: Load, process, and export images (PNG/JPEG)
  - Remove background (near-white pixel knockout)
  - Multiple resolution options
  - Transparency preservation

- **Data Export**: Structured data management
  - CSV editing and export
  - Sample row generation
  - Local processing (no data sent anywhere)

## Deployment

Hosted on GitHub Pages: https://huseyinozdes.github.io/randomwhattheflyingduck/

## Technology

- **Frontend**: HTML5, CSS3 (Tailwind), Vanilla JavaScript
- **Styling**: Space Mono font (Google Fonts)
- **Processing**: Canvas API, File API, Blob API
- **License**: GPL-3.0

## Local Development

Simply open `index.html` in your browser. No build process or dependencies required.

## Privacy

All image processing and data manipulation runs entirely in your browser. No data is sent to any server.
